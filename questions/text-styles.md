## What stuff can I do to text?

Here's a bunch of styles!

```css
p {
  /* change the font */
  font-family: "Garamond", serif;

  /* change the font size */
  font-size: 20px;

  /* make it "italic," or "normal" */
  font-style: italic;
  
  /* make it bold, or normal */
  font-weight: bold;
  
  /* change the spacing between lines */
  line-height: 1.5; 

  /* change the spacing between letters */
  letter-spacing: 2px;

  /* change the font color */
  color: turquoise;

  /* add a text shadow slightly below and right of the text, blur it slightly, and make the shadow slightly blue */
  text-shadow: 1px 1px 1px rgba(0,0,255,0.25);

  /* change it to "left," "right," or "center" */
  text-align: right;

  /* change it to "uppercase," "lowercase," or "capitalize" */
  text-transform: uppercase;

  /* can be "underline" or "strikethrough" */
  text-decoration: underline;
}
```
